# Быстрая сводка: Исправление дат в вебхуках RentProg

**Дата:** 2025-11-06  
**Статус:** ✅ Завершено

---

## Что было сделано

### Проблема
```
Error: date/time field value out of range: "14-11-2025 21:00"
```

RentProg отправляет даты в формате `DD-MM-YYYY HH:mm`, PostgreSQL ожидает ISO формат.

### Решение

Добавлена автоматическая конвертация дат в ноду **Prepare Create** всех 4 процессоров:

- ✅ Tbilisi Processor Rentprog
- ✅ Batumi Processor Rentprog  
- ✅ Kutaisi Processor Rentprog
- ✅ Service Center Processor Rentprog

### Как работает

```javascript
"12-11-2025 09:00" → "2025-11-12T09:00:00+04:00"
```

Конвертирует все поля с датами автоматически перед записью в БД.
