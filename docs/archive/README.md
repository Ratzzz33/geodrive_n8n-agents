# Архив устаревших документов

Этот каталог содержит устаревшие документы, которые были перемещены из корня проекта для улучшения читаемости.

## Структура

- `final/` - Файлы с названием FINAL (завершенные задачи)
- `complete/` - Файлы с названием COMPLETE (завершенные задачи)
- `status/` - Файлы с названием STATUS (статусы задач)
- `reports/` - Файлы с названием REPORT (отчеты)

## Когда можно удалить

Эти файлы можно безопасно удалить через 3-6 месяцев после архивации, если они не содержат критически важной информации.

## Восстановление

Если нужен какой-то документ из архива, просто переместите его обратно в корень проекта.
