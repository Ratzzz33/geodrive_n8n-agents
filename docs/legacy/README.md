# Legacy документация

В этой директории находятся материалы, которые относятся к прежней инфраструктуре на базе Netlify. После миграции на Nginx (`n8n.rentflow.rentals` / `webhook.rentflow.rentals`) они оставлены исключительно для исторической справки и больше не применяются в рабочем процессе.

## Содержимое
- `netlify/` — инструкции и настройки, связанные с Netlify (токены, переменные окружения, старые гиды по вебхукам).

### Как читать
- Используйте эти файлы только для анализа прошлых решений.
- **Не копируйте** команды и настройки из legacy-документации в текущую конфигурацию.

Актуальные инструкции по среде находятся в корневых документах (`claude.md`, `config/N8N_VARIABLES.md`, `SESSION_REPORT_NGINX_NETLIFY_MIGRATION.md`).
